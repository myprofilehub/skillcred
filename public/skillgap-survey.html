Understanding SkillCred's Architecture & Upskilling Model

SkillCred is a project-based tech bootcamp platform built specifically for engineering students and fresh graduates in India. It bridges the gap between what colleges teach and what the industry actually needs — by making learners build and ship real products from Day 1.

🎯 Target Audience
  • Engineering students (2nd yr and above)
  • Fresh graduates (0–2 years out)
  • Career switchers entering tech
📦 Program Formats
  • 4-week Intensive (focused stream)
  • 8-week Deep Dive (extended capstone)
  • Online-first, hybrid available
🏛️ Platform Pillars
  • Stream-specific curriculum (8 tracks)
  • Project-based learning by building
  • PAT certification (not exams)
  • HR Connect hiring pipeline
📅 Learning Schedule
  • Mon–Sat, 2-hour daily sessions
  • 192 sessions across all streams
  • Mentor-guided live + async
💡 SkillCred is not a MOOC. It is a structured, mentor-led bootcamp where the output is a working product — not a quiz score.

Each stream is a self-contained 4-week curriculum with its own project set, mentor pool, and PAT assessment. Learners choose one stream per cohort.

📌 Each stream has 6 projects: 2 solo builds, 1 pair project, and 3 capstone options. Learners pick their capstone based on interest and career goal.

SkillCred follows a Build First, Learn As You Go philosophy. Instead of 3 weeks of theory followed by one project, every concept is introduced through a problem the learner is immediately trying to solve.

Week 1
Foundation Sprint
Core tools, environment setup, fundamentals through a guided Solo Project #1. Mentor introduces concepts only when the project demands it.
📦 Output: Solo Project 1 — deployed & documented
Week 2
Depth & Integration
Intermediate concepts, APIs, databases, and tool integration. Learner builds Solo Project #2 with increasing independence from mentor.
📦 Output: Solo Project 2 — peer-reviewed
Week 3
Collaboration Sprint
Pair project — two learners build together with defined roles (lead dev + reviewer). Introduces Git workflow, code review, and division of work.
📦 Output: Pair Project — joint GitHub repo
Week 4
Capstone & PAT
Group capstone project (3–4 learners) chosen from 3 options per stream. Final week ends with the Project Assessment Test — defense, demo, and live extension round.
🏆 Output: Capstone + PAT Certification
⚙️ Mon–Sat, 2-hour live sessions. Theory is never taught in isolation — every concept is tied to something the learner is actively building that day.

The PAT is SkillCred's alternative to written exams. A learner is assessed not on what they memorised, but on what they built, why they built it that way, and how they adapt to new problems.

01
📁
Project Submission
Learner submits working project with GitHub repo, README, deployed link, and a 2-minute demo video explaining what they built and the key decisions made.
02
🔬
Technical Defense
Industry mentor or assessor probes the code — architecture choices, edge cases, why certain libraries were used, what would break under scale. 15–20 min session.
03
⚡
Live Problem Extension
A new feature or constraint is introduced on-the-spot (e.g. "Add authentication to this" or "Make it work offline"). Learner has 20 minutes to approach the solution live.
04
🏅
PAT Score & Certification
Assessor scores across 4 dimensions: Technical Depth, Code Quality, Problem Adaptability, and Communication. Learner receives a stream-specific PAT certificate with scores.
